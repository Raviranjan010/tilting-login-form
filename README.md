# Tilting Login Form

An animated login interface that features:

* Dynamic 3-D tilt following cursor movement.
* Gradient-animated glassmorphism card design.
* Neon gradient button with wavy text hover and async mock login.
* Floating colorful particles background.

Pure **HTML**, **CSS**, and **Vanilla JavaScript** with zero dependencies.

## File Structure
```
tilting-login-form/
├── index.html   # Markup linking external assets
├── style.css    # All styling & keyframes animations
├── script.js    # Particles, tilt, and button logic
└── README.md    # Documentation (this file)
```

## Getting Started
1. Clone/download the repository.
2. Open `index.html` in a modern browser – no build steps.

## Customization
* Change background colors by editing `body` gradient in `style.css`.
* Adjust particle count via `particleCount` in `script.js`.
* Modify input placeholders, texts and social icons directly in `index.html`.
